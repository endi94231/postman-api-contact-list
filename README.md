# postman-api-contact-list
API automation testing using postman

This project provides an example for testing REST APIs using Postman with sample APIs Documentation: https://documenter.getpostman.com/view/4012288/TzK2bEa8#e388e723-b3e6-4a9b-99d5-609cbd6d4a4d

NB This project only implements basic testing of the API such as building test suites and testing expected response.

Steps to Try:
1. Download and install Postman API: https://www.postman.com/downloads.
2. Clone Project.
3. Import *ContactListAppCollections.postman_collection* and *ContactListAppEnvironments.postman_environment*.
   - Click button "Import".
   - Select collection and environment file has been cloned.
   - Click button "Import".
4. Run collection *ContactListAppCollections*.
   - Click collection "ContactListAppCollections".
   - Click button "Run".
   - Click button "ContactListAppCollection".
